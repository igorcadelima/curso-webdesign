Estruturas de decisão

Utilizam-se estruturas de decisão quando desejamos executar determinado conjunto de instruções para uma dada condição.

Em JS temos as seguintes declarações condicionais:

  • if
  • else if
  • else

Estruturas de decisão: if

O comando condicional if simples serve para executar todo o código que esteja entre chaves após a condição do comando, caso o resultado lógico da condição seja true.

Exemplo:

var idade = prompt("Qual sua idade?");
if (idade >= 18) {
  document.write("Você é maior de idade.");
}
document.write("<br/>Fim de página.");

Estruturas de decisão: if ... else

O comando else permite adicionar um bloco de instruções alternativo a ser executado caso a o resultado lógico da condição do if resulte em false.

Exemplo:

var idade = prompt("Qual sua idade?");
if (idade >= 18) {
  document.write("Você é maior de idade.");
} else {
  document.write("Você é menor de idade.");
}
document.write("<br/>Fim de página.");

Estruturas de decisão: if ... else if ... else

Com else if pode-se adicionar outras condições à estrutura.

Exemplo:

var valor = prompt("Digite um valor: ");
if (valor > 10) {
  document.write("O valor digitado é maior que 10.");
} else if (valor < 10) {
  document.write("O valor digitado é menor que 10.");
} else {
  document.write("O valor digitado é igual a 10.");
}

Estrutura de repetição: for

A estrutura for é a mesma coisa que a instrução para vista nas aulas de lógica de programação.

Esse tipo de estrutura deve ser utilizada quando já se conhece a quantidade de vezes que um determinado bloco de instruções deve ser repetido.

Sintaxe:

for (<expressão inicial>; <condição>; <atualização>) {
  <instruções>;
}

Estrutura de repetição: for - exemplo

Para entender melhor a estrutura for, execute o seguinte código: 

for (var i = 0; i <= 5; i++) {
  document.write("O valor de i = ", i);
  document.write("<hr>");
}

Estrutura de repetição: while

A estrutura while é a mesma coisa que a instrução enquanto ... faca vista nas aulas de lógica de programação.

while executa as instruções especificas em seu bloco de código enquanto a condição for verdadeira.

Sintaxe:

while (<condição>) {
  <instruções executadas enquanto condição for verdadeira>
}

Estrutura de repetição: while - exemplo

A seguir está o código para o mesmo exemplo anterior reescrito utilizando while:

var i = 0;
while (i <= 5) {
  document.write("Valor de i = ", i);
  document.write("<hr>");
  i++; //i = i + 1;
}

Estrutura de repetição: do while

A estrutura do ... while é a mesma coisa que a instrução repita ... ate vista nas aulas de lógica de programação.

Diferente da estrutura while, um loop do ... while uma vez antes que a condição seja avaliada.

Sinstaxe:

do {
  <bloco de instruções>
} while (<condição>);

Estrutura de repetição: do ... while

O exemplo a seguir demonstra que, mesmo quando a primeira verificação da condição é falsa, as instruções dentro do loop são executadas pelo menos uma vez.

var i = 6;
do {
  document.write("Valor de i = ", i);
  document.write("<hr>");
  i++; //i = i + 1;
} while (i <= 5)

A instrução break

A instrução break encerra a execução do loop delimitador mais próximo (do, for ou while) na qual ele aparece.

Tem por objetivo o cancelamento da execução do loop sem que haja verificação na condição de saída do loop.

Exemplo:

var i = 0;
while (i <= 5) {
  i++; //i = i + 1;
  document.write("Valor de i = ", i);
  document.write("<hr>");
  break;
}

A instrução break

Uma vantagem do uso do break é que ele pode parar o loop dentro de uma condição criada por um if.

Exemplo:

var i = 0;
while (i <= 5) {
  i++; //i = i + 1;
  if (i == 3) {
    break;
  }
  document.write("Valor de i = ", i);
  document.write("<hr>");
}

A instrução continue

A instrução continue transmite o controle para a próxima iteração da instrução do, for ou while delimitadora mais próxima em que aparece.

Exemplo:

var lista = [1, 33, 5, 2, 5, 93];
for (var i = 0; i <=5; i++) {
  if (lista[i] == 5) {
    continue;
  }
  document.write("O valor de lista[", i,"] é: ", lista[i]);
  document.write("<hr>");
}
