Funções

  • Uma função é um bloco de código designado para realizar uma tarefa em particular.
  • Além de evitar repetição desnecessária de código, funções facilitam manutenção e leitura de código.
  • alert, prompt, e typeof são exemplos de funções que usamos até agora.

Criando funcões

As duas maneiras mais comuns de criar funções em JS são as seguintes: declaração de função e expressão de função.

Declaração de função:

function nome(parametro1, parametro2, ..., parametroN){
  // Código a ser executado
}

Expressão de função:

var nome = function(parametro1, parametro2, ..., parametroN) {
  // Código a ser executado
}

Parâmetros são opcionais.

Funções

Podemos classificar funções como:

  • Funções sem retorno (procedimento);
  • Funções com retorno;
  • Funções com parâmetros;
  • Funções com parâmetros e retorno.

Funções sem retorno - exemplo

No arquivo de script JS adicione:

function saudacao() {
  alert("Olá, visitante!");
}

No corpo do arquivo HTML adicione:

<a href="#" onClick="saudacao()"> Abrir alerta </a>

Funções com retorno - exemplo

Funções com retorno usam a instrução return para retornar algo da função.

function calculaIMC() {
  var altura = 1.80;  // Altura em metros
  var peso = 80;      // Peso em quilogramas
  var imc = peso / (altura * altura);
  return imc;
}

document.write("O índice de massa corporal (IMC) é: ", calculaIMC());

Funções com parâmetros - exemplo

Esse tipo de função nos permite passar valores/argumentos para a função para que se possa utilizá-los dentro da função.

Um função pode receber vários argumento desde que sejam separados por vírgulas.

function saudacao (nome) {
  alert("Olá, " + nome + "!");
}

var nomeUsuario = prompt("Qual o seu nome?");
saudacao(nomeUsuario);

Funções com parâmetros e retorno - exemplo

Como um nome sugere, esse tipo de função permite que valores sejam passados como argumentos e que valores possam ser retornados.

function calcArea (base, altura) {
  var area = base * altura;
  return area;
}

var areaRetangulo = calcArea(2, 4);
document.write("A área do retângulo é: ", areaRetangulo);

Eventos

Pode-se dizer que evento é tudo que um usuário realiza quando está visitando uma página web. Ex.: mover ou clicar com o mouse sobre um objeto de texto, de imagem, ou de formulário.

Outros exemplos:

  • Carregamento de uma página;
  • Passagem do mouse sobre algum elemento HTML;
  • Seleção de um campo de entrada em um formulário HTML;
  • Submissão de um formulário HTML;
  • Pressionamento de tecla;
  • e muitos outros.

Captura de eventos

Baseado no que foi visto até agora, serão usadas declarações inline e incorporadas de código JS para capturar eventos.

Por exemplo:

<head>
  <meta charset="UTF-8">
  <title>Exemplo</title>
  <script>
    function acao() {
      alert("Opa! Você acabou de clicar no campo de email.");
    }
  </script>
</head>
<body>
  Email: <input name="email" type="text" onClick="acao();">
</body>

Tipos de eventos

Esses serão os eventos vistos nessa aula:

Evento Descrição
onClick Acontece quando um objeto é clicado.
onMouseOver Acontece quando o cursor é colocado sobre um objeto.
onMouseOut Acontece quando o curso deixade estar sobre um objeto.
onLoad Acontece quando um objeto é carregado.
onSelect Acontece quando um campo de formulário é selecionado.
onSubmit Acontece quando a ação de submissão de um formulário é iniciada.
onFocus Acontece quando um elemento de formulário recebe foco.
onBlur Acontece quando um elemento de formulário perde foco.
onChange Acontece quando um objeto é modificado.

Lista mais extensa de eventos aqui.

onClick

onClick é um dos eventos mais utilizados em JS pois é acionado sempre que acontece um clique do mouse sobre um elemento HTML como botão, campo de texto, etc.

Exemplo:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <title>Exemplo onClick</title>
  <script>
    function acao() {
      alert("A forma padrão para email é nome@servidor.com.");
    }
  </script>
</head>
<body>
  Email: <input name="email" type="text" onClick="acao();">
</body>
</html>

onClick - exemplo

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <title>Exemplo onClick</title>
  <script>
    function mudaBackground(cor) {
      document.bgColor = cor;
    }
  </script>
</head>
<body>
  <h3>Mude a cor de fundo da página:</h3>
  <ul>
    <li><a href="#" onclick="mudaBackground('red')">Vermelho</a></li>
    <li><a href="#" onclick="mudaBackground('green')">Verde</a></li>
    <li><a href="#" onclick="mudaBackground('blue')">Azul</a></li>
    <li><a href="#" onclick="mudaBackground('white')">Padrão</a></li>
  </ul>
</body>
</html>

onLoad - exemplo

...
  <script>
    function perguntaNome() {
      var nome = prompt("Qual seu nome?");
      return nome;
    }

    function escreveNome() {
      var nome = perguntaNome();
      document.write("Bom dia, ", nome, ".");
    }
  </script>
...
  <body onLoad="escreveNome();"></body>

onSelect - exemplo

...

onChange - exemplo

...
  <script>
    function alteracao(id) {
      var alterado = document.getElementById(id).value;
      console.log("O campo " + id + " foi alterado.");
      console.log("Alteração: " + alterado);
    }
  </script>
...
  <body>
    <form>
      Login: <input id="login" type="text" onChange="alteracao('login');"> <br />
      Senha: <input id="senha" type="password" onChange="alteracao('senha');">
    </form>
  </body>

onSubmit

...
  <script>
    function submissao() {
      var login = document.getElementById('login').value;
      var senha = document.getElementById('senha').value;
      if (login == "") {
        alert("Digite seu login.");
      }
      if (senha == "") {
        alert("Digite sua senha.");
      }
    }
  </script>
...
  <body>
    <form onSubmit="submissao();">
      Login: <input id="login" type="text"> <br />
      Senha: <input id="senha" type="password"> <br />
      <input type="submit" value="Enviar">
    </form>
  </body>
