Aula 05 - Estrutura de dados homogênea: vetores

Igor Conrado Alves de Lima - prof.igor@microcamp.com.br

Problema motivacional

Calcular e exibir a média geral de uma turma de 8 alunos. A média a ser obtida deve ser a média geral das médias de cada aluno obtida durante o ano letivo. Para representar a média do primeiro aluno será utilizada a variável MD1, para o segundo MD2 e assim por diante.

Motivação

Até agora nós somos capazes de armazenar diferentes valores em diferentes variáveis. No exemplo anterior, por exemplo, tivemos que criar oito variáveis para armazenar as médias dos alunos.

Linguagens de programação nos oferecem uma estrutura de dados conhecida como vetores que nos permite utilizar apenas uma variável para armazenar diversos valores do mesmo tipo.

Vetores

Vetor é uma técnica de programação que permite trabalhar com o agrupamento* de várias informações dentro de uma mesma variável.

Vetores também são comumente conhecidos como arranjos, matrizes, ou arrays.

*Esse agrupamento deve ser realizado obedecendo o mesmo tipo de dado. Daí o nome estrutura de dado homogênea.

Atribuição de um vetor

A criação de vetores em Português Estruturado tem a seguinte sintaxe:

        VARIÁVEL: VETOR[<dimensão>] de <tipo de dado>

Atribuição de um vetor

Um vetor para o exemplo anterior poderia ser criado da seguinte forma:

        MD: VETOR[1..8] de real

Sendo seu nome MD e seu tamanho de 1 a 8.

Problema

Calcular e exibir a média geral de uma turma de 8 alunos usando vetor. A média a ser obtida deve ser a média geral das médias de cada aluno obtida durante o ano letivo. Para representar a média dos alunos será utilizado o vetor MD.

Comparação entre uso de variáveis e vetor

Armazenamento de médias utilizando vetores e variáveis simples:

            MD[1] = 4.5 
            MD[2] = 6.5 
            MD[3] = 8.0 
            MD[4] = 3.5 
            MD[5] = 6.0 
            MD[6] = 7.0 
            MD[7] = 6.5 
            MD[8] = 6.0
          
            MD1 = 4.5 
            MD2 = 6.5 
            MD3 = 8.0 
            MD4 = 3.5 
            MD5 = 6.0 
            MD6 = 7.0 
            MD7 = 6.5 
            MD8 = 6.0

Os números entre colchetes são chamados índices.

Comparação entre uso de variáveis e vetor

Usando vetores nós temos:

  • Código mais legível;
  • Maior escalabilidade.

Hora de exercitar!

Exercício 1

Desenvolver um programa que efetue a leitura de dez elementos de um vetor A. Construir uma matriz B de mesmo tipo, observando a seguinte lei de formação: se o valor do índice for par, o valor deverá ser multiplicado por 5, sendo ímpar, deverá ser somado com 5. Ao final mostrar o conteúdo da matriz B.

Lembre que um número será par se quando dividido por 2 obtiver resto igual a 0.

Exercício 2

Desenvolver um programa que efetue a leitura de cinco elementos de um vetor A. No final, apresente o total da soma de todos os elementos que sejam ímpares.

Lembre que um número será ímpar se quando dividido por 2 obtiver resto igual a 1.
