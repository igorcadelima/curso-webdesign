O que é PHP?

  • PHP é um acrônimo recursivo para "PHP: Hypertext Preprocessor", originalmente Personal Home Page.
  • Linguagem de programação usada para criar conteúdo dinâmico para a web.
  • PHP é executada no servidor e devolve para o lado cliente somente o código HTML.

Ferramentas utilizadas - Preparando o servidor

Para preprar o nosso servidor e começar a programar em PHP utilizando banco de dados MySQL nós precisaremos do seguinte:

  • Interpretador PHP;
  • Um servidor web (Apache);
  • Servidor de banco de dados (MySQL).

O que é o XAMPP?

  • Um pacote de distribuição de programas de desenvolvimento web.
  • Ao baixar e instalar o XAMPP, o mesmo instala e configura automaticamente o Apache e o MySQL.
  • Acessando esse link do site é possível baixar o pacote para nossa máquina.
  • Ao executar o instalador, a instalação deve ser feita automaticamente e sem complicações.

Executando o XAMPP Control Panel

  1. Ao instalar o XAMPP, o mesmo pode ser ativado pelo ícone "XAMPP Control Panel".
  2. Clique o botão direito sobre o ícone XAMPP Control Panel e então Executar como administrador.
  3. Inicialize os servidores Apache e MySQL clicando em Start.
  4. Verifique a instalação do Apache acessando o endereço http://localhost.

Criando páginas dinâmicas em PHP

Informações básicas:

  • Por padrão, o XAMPP é instalado no caminho C:\xampp.
  • Todas as páginas que você criar deve estar dentro do diretório htdocs.
  • Dentro de htdocs você deverá criar um diretório para cada projeto.
  • Atenção: os nomes dos diretórios e arquivos não devem conter espaços.

"Hello, world!"

Digite o código abaixo no seu editor de texto e salve como exercicio1.php dentro do diretório "aulasphp" dentro de htdocs:

<?php
  echo "Hello, world!";
?>

Execute o código abrindo o link http://localhost/aulasphp/exercicio1.php em seu navegador de internet.

Entendendo o código

  1. Para que o servidor interprete o código php, o mesmo deve estar dentro da tag <?php ... ?>
  2. A instrução/comando echo tem a função de escrever na tela do navegador, ou seja, devolve em formato HTML o código php processado no servidor.
<?php
  echo "Hello, world!";
?>

Comentário de uma linha

Para criar comentários de apenas uma linha em PHP utilizamos // (duas barras).

Tudo que seguir // em uma linha de código será ignorado pelo interpretador PHP.

exemplo2.php

<?php
  // Isto é um comentário de uma linha que será ignorado pelo interpretador PHP.
  echo "Você está lendo uma página dinâmica PHP."
?>

Comentário de várias linhas

Podemos digitar várias linhas de comentário desde que estejam entre /* e */.

exemplo3.php

<?php
  /*
    Isto é um comentário de várias linhas.
    Tudo que for digitado aqui, mesmo que em várias linhas,
    será ignorado pelo interpretador PHP.
  */
  echo "Você está lendo uma página dinâmica PHP."
?>

PHP e HTML

Não precisamos usar o comando echo para escrever um documento HTML por inteiro.

Podemos facilmente misturar nosso código PHP dentro de uma página HTML.

exercicio4.php

<!DOCTYPE html>
<html>
<head>
  <title>Exercício 4 de PHP</title>
</head>
<body>
  <p>Este é o primeiro parágrafo.</p>
  <p><?php echo "Este é o segundo parágrafo."; ?></p>
  <p>Este é o terceiro paragráfo.</p>
  <?php echo "<p>Este é o quarto parágrafo.</p>"; ?>
</body>
</html>
