Constantes

  • Uma constante é um identificador para um único valor imutável.
  • O valor de uma constante não pode ser alterado durante a execução do programa.
  • Nomes de constantes devem conter caracteres especiais ou espaços.
  • É boa prática começar nomes de constantes com letra ou underline (_).
  • Por convenção, identificadores de constantes são sempre escritos com letras maiúsculas.

Constantes

exercicio1.php

<?php
  // Nomes válidos de constantes
  define("BOLA1", "Futebol");
  define("BOLA2", "Basquete");
  define("BOLA_3", "Ping-pong");

  echo BOLA1 . "<br>";	// O operador . é usado para concatenar as strings
  echo BOLA2 . "<br>";
  echo BOLA_3;
?>

Variáveis

Ao contrário das constantes, valores de variáveis podem ser alterados em qualquer momento da execução do programa.

Regras de variáveis PHP:

  • Uma variável começa com o caractere $ (cifrão), seguido pelo nome da variável
  • O nome de uma variável deve começar com uma letra ou underline
  • O nome de uma variável não pode conter caracteres especiais (ex. á, ç, õ, @, etc) ou espaços, apenas caracteres alfa-numéricos e underlines
  • Nomes de variáveis são sensíveis à capitalização (case-sensitive) (ex. $nome e $NOME são duas variáveis diferentes)

Variáveis

exercicio2.php

<?php
  $valor1 = 100;	// Todas as letras dessa variável são minúsculas
  $Valor1 = 200;	// Apenas a primeira letra dessa variável é maiúscula
  $VALOR1 = 300;	// Todas as letras dessa variável são maiúsculas

  echo "$valor1 - $Valor1 - $VALOR1";	// 100 - 200 - 300
?>

Variáveis

exercicio3.php

<?php
  $valorUm	= 1500;	
  $valorDois 	= 1100;

  $idadeFuncionario 	= 36;
  $idadeCliente 	= 26;

  echo "A idade do cliente é $idadeCliente e a do funcionário é $idadeFuncionario.<br>";
  echo "O valor do produto que o cliente quer comprar é $valorUm, mas ele só quer pagar $valorDois.";
?>

Tipos de dados em PHP

Existem 8 tipos de dados primitivos em PHP:

  • integer
  • float (ou double)
  • string
  • boolean
  • array
  • object
  • resource
  • NULL

Mais detalhes sobre resource.

O tipo inteiro (integer)

Um inteiro é qualquer número sem casas decimais, positivo ou negativo.

exemplo4.php

<?php
  // Declaração de uma variável como inteiro
  $idadeAluno = 18;
?>
<!DOCTYPE html>
<html>
<head>
  <title>Tipo inteiro em PHP</title>
</head>
<body>
  <p>O aluno tem <?php echo $idadeAluno; ?> anos de idade.</p>
</body>
</html>

O tipo float ou double

Também chamado de número de ponto flutuante, um float (ou double) é qualquer número real.

exemplo5.php

<?php
  // Definição de uma constante como float
  define("PI", 3.14159265);
?>
<!DOCTYPE html>
<html>
<head>
  <title>Tipo float em PHP</title>
</head>
<body>
  <p>O valor de pi é <?php echo PI; ?>.</p>
</body>
</html>

O tipo string

Uma string é uma série/cadeia de caracteres.

Para declarar strings podemos usar aspas simples ('') ou aspas duplas ("").

exemplo6.php

<?php
  $nome1 = "Mario";	// string criada com aspas duplas
  $nome2 = 'Luigi';	// string criada com aspas simples

  echo "<h2>A dupla dinâmica: $nome1 e $nome2!</h2>";
?>

String com aspas simples

String com aspas simples serão exibidas quase como elas realmente são.

Variáveis não são interpretadas.

Para exibir uma aspa simples é necessário escapá-la com uma contra-barra (\') e para exibir uma contra-barra é necessário escapá-la com outra contra-barra (\\).

exemplo7.php

<?php
  $texto = 'Sonic "atravessou" a caixa d\'água.';	// contra-barra usada para exibir aspa simples

  echo 'A variável $texto não foi interpretada.<br>';
  echo $texto;
?>

String com aspas duplas

String com aspas duplas serão exibidas com os valores das variáveis interpretados.

Para exibir uma aspa dupla é necessário escapá-la com uma contra-barra (\") e para exibir uma contra-barra é necessário escapá-la com outra contra-barra (\\).

exemplo8.php

<?php
  $texto = "Sonic \"atravessou\" a caixa d'água.";	// contra-barra usada para exibir aspas duplas

  echo "O conteúdo da variável é: $texto";
?>

O tipo booleano (boolean)

Esse tipo de dados apenas aceita os valores TRUE (verdadeiro) ou FALSE (para falso).

O tipo booleano é case-insensitive (insensível à capitalização). No entanto, prefira usar ou TRUE e FALSE ou true e false.

exemplo9.php

<?php
  $verdadeiro 	= TRUE; 	// True, true, TrUe, etc. seria a mesma coisa.
  $falso 	= FALSE;	// False, false, FaLsE, etc. seria a mesma coisa.
?>

O tipo nulo (NULL)

NULL é o único valor possível do tipo NULL.

O valor especial NULL indica que a variável não tem valor.

O tipo NULL também é case-insensitive mas prefira sempre utilizar ou NULL ou null.

exemplo10.php

<?php
  $nulo = NULL;	// Declaração de uma variável nula
  echo $nulo;
?>
