Estruturas de decisão

Utilizam-se estruturas de decisão quando desejamos executar determinado conjunto de instruções para uma dada condição.

Em PHP temos as seguintes declarações condicionais:

  • if
  • elseif
  • else

Desvio condicional simples: if

if serve para executar todo o código de instrução localizado entre as chaves do comando após a condição caso o resultado da avaliação lógica seja verdadeiro (true).

exercicio1.php:

<?php
  $nota = 10;

  if ( $nota >= 7 ) {
    echo 'O aluno foi aprovado.';
  }
?>

Desvio condicional composto: if ... else

O comando else é utilizado logo após a estrutura de controle if ou elseif.

Sempre que as condições if e elseif da mesma estrutura retornarem false o comando else será executado.

exercicio2.php

<?php
  $nota = 5;

  if ( $nota >= 7 ) {
    echo 'O aluno foi aprovado.';
  } else {
    echo 'O aluno foi reprovado.';
  }
?>

Outro desvio condicional composto: if ... elseif ... else

Com elseif pode-se adicionar outras condições à estrutura.

Uma condição elseif só será executada caso as condições anteriores a ela sejam avaliadas como falsas (false).

exercicio3.php:

<?php
  $nota = 6;

  if ( $nota >= 7 ) {
    echo 'O aluno foi aprovado.';
  } elseif ( $nota >= 5) {
    echo 'O aluno está de recuperação.';
  } else {
    echo 'O aluno foi reprovado.';
  }
?>

Crie um programa que verifique se o valor armazenado na variável totalCompra e, baseado no valor armazenado, escreva na página quanto de desconto o cliente tem direito. As condições são as seguintes:

  • escreva "Você tem direito a 5% de desconto." caso o valor seja menor que 100;
  • escreva "Você tem direito a 10% de desconto." caso o valor seja maior ou igual a 100 e menor que 300;
  • escreva "Você tem direito a 15% de desconto." caso o valor seja maior ou igual a 300.

Salve seu programa como exercicio4.php.

A estrutura de decisão switch ... case ... default

A estrutura switch é parecida com uma série de if e elseifs numa mesma estrutura.

O switch permite que uma variável seja testada comparando a igualdade de seu valor (==) com uma lista de diferentes casos.

<?php
  // exercicio4.php
  $opcao = 2;
  switch ($opcao) {
    case 0:
      echo "A opção escolhida foi 0.";
      break;
    case 1:
      echo "A opção escolhida foi 1.";
      break;
    case 2:
      echo "A opção escolhida foi 2.";
      break;
  }
?>

A estrutura de decisão switch ... case ... default

O código equivalente do exercicio4.php usando a estrutura if seria o seguinte:

<?php
  // exercicio5.php
  $opcao = 0;

  if ( $opcao == 0 ) {
    echo "A opção escolhida foi 0.";

  } elseif ( $opcao == 1 ) {
    echo "A opção escolhida foi 1.";

  } elseif ( $opcao == 2 ) {
    echo "A opção escolhida foi 2.";

  }
?>

A estrutura de decisão switch ... case ... default

Outro exemplo utilizando a estrutura switch.

<?php
  // exercicio6.php
  $fruta = "maçã";
  switch ($fruta) {
    case "maçã":
      echo '$fruta é maçã.';
      break;
    case "uva":
      echo '$fruta é uva.';
      break;
    case "laranja":
      echo '$fruta é laranja.';
      break;
  }
?>

Entendendo a estrutura switch

  • A estrutura é executada linha por linha (na verdade, instrução por instrução)
  • Nada é executado no começo. Somente quando um case com um valor correspondente à variável analisada é encontrado é que as instruções serão executadas.
  • PHP continua executando as instruções até chegar ao fim da switch ou encontrar a instrução break.

Vamos tentar executar o exercício anterior sem usar break:

<?php
  // exercicio7.php
  $fruta = "maçã";
  switch ($fruta) {
    case "maçã":
      echo '$fruta é maçã.';
    case "uva":
      echo '$fruta é uva.';
    case "laranja":
      echo '$fruta é laranja.';
  }
?>

O caso padrão (default)

Se nenhum case com um valor correspondente à variável analisada é encontrado, então um caso padrão pode ser executado. Esse caso padrão é criado usando a instrução default.

<?php
  // exercicio8.php
  $opcao = 2;
  switch ($opcao) {
    case 0:
      echo "A opção escolhida foi 0.";
      break;
    case 1:
      echo "A opção escolhida foi 1.";
      break;
    case 2:
      echo "A opção escolhida foi 2.";
      break;
    default:
      echo "Opção escolhida não foi 0, 1, ou 2.";
  }
?>
