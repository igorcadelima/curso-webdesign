Estrutura de repetição for

Essa estrutura é normalmente utilizada quando conhecemos quantas vezes um bloco de instruções deverá ser executado repetidamente.

Sua sintaxe é a seguinte:

for (<inicialização>; <condição>; <incremento>) {
  // instruções
}

Estrutura de repetição for

O comando for possui basicamente 3 parâmetros:

  1. Uma variável numérica com valor inicial para ser incrementado ou decrementado.
  2. Uma condição que para controlar o número de iterações no loop.
  3. Uma forma de incremento para a variável numérica criada.

Exemplo:

<?php
  // exercicio1.php
  for ($i = 0; $i < 10; $i++) {
    echo "A iteração atual é: $i <br>";
  }
?>

Estrutura de repetição while

  • A estrutura while é a mesma coisa que a instrução enquanto ... faca vista nas aulas de lógica de programação.
  • while executa as instruções especificas em seu bloco de código enquanto uma dada condição de controle for TRUE.
  • Se a condição for avaliada como FALSE, o processamento é desviado para fora do loop.

Sintaxe:

while (<condição>) {
  <instruções executadas enquanto condição for verdadeira>
}

Estrutura de repetição while

<?php
  // exercicio2.php
  $i = 0;
  while ($i < 10) {
    echo "A iteração atual é: $i <br>";
    $i++; // $i = $i + 1;
  }
?>

Estrutura de repetição do ... while

A estrutura do ... while é a mesma coisa que a instrução repita ... ate vista nas aulas de lógica de programação.

Diferente da estrutura while, um loop do ... while é executado uma vez antes que a condição seja avaliada.

Sintaxe:

do {
  <bloco de instruções>
} while (<condição>);

Estrutura de repetição do ... while

<?php
  // exercicio3.php
  $i = 0;
  do {
    echo "A iteração atual é: $i <br>";
    $i++; // $i = $i + 1;
  } while ($i >= 10);
?>

A instrução break

A instrução break encerra a execução do loop delimitador mais próximo (do, for ou while) na qual ele aparece.

Tem por objetivo o cancelamento da execução do loop sem que haja verificação na condição de saída do loop.

Exemplo:

<?php
  // exercicio4.php
  $i = 0;
  while ($i < 10) {
    echo "O valor de i é: $i <br>";
    i++; //i = i + 1;
    break;
  }
?>

A instrução break

Uma vantagem do uso do break é que ele pode parar o loop dentro de uma condição criada por um if, por exemplo.

Exemplo:

<?php
  // exercicio5.php
  for ($i = 0; $i < 10; $i++) {
    echo "A valor de i é: $i <br>";
    if ($i == 4) {
      break;
    }
  }
?>

A instrução continue

A instrução continue transmite o controle para a próxima iteração da instrução do, for ou while delimitadora mais próxima em que aparece.

Exemplo:

<?php
  // exercicio6.php
  $lista = array(1, 33, 5, 2, 5, 93);
  for ($i = 0; $i <=5; $i++) {
    if ($lista[$i] == 5) {
      continue;
    }
    echo "O valor de lista[$i] é $lista[$i] <br>";
  }
?>

Estrutura de repetição foreach

A estrutura foreach fornece uma maneira mais fácil de iterar sobre matrizes.

foreach funciona somente com arrays e objetos.

Existem duas sintaxes:

foreach ($matriz as $valor) {
  // instruções
}
foreach ($matriz as $chave => $valor) {
  // instruções
}

Estrutura de repetição foreach

<?php
  // exercicio7.php
  $nomes = array("Guinho", "Zezinho", "Luisinho");

  foreach ($nomes as $nome) {
    echo "O nome da vez é $nome.<br>";
  }
?>

Estrutura de repetição foreach

<?php
  // exercicio8.php
  $doc = array();
  $doc['nome']  = "Igor";
  $doc['fone']  = "(82)99999-0000";
  $doc['email'] = "prof.igor@microcamp.com.br";
  
  foreach ($doc as $chave => $valor) {
    echo "$chave - $valor<br>";
  }
?>
