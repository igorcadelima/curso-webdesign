Funções

  • Bloco de código reusável com um nome que é designado para realizar uma tarefa em particular.
  • Economiza tempo e trabalho para ações que irão se repetir.

Sintaxe:

function <nome_da_funcao> ( <parâmetros>, ...) {
  // Instruções
}

Funções

Funções são muito importante em sistemas grandes que requerem execução de tarefas rotineiras como, por exemplo, calcular o dobro de um número.

<?php
  // exercicio1.php

  function calcula_dobro($numero) {
    // Faz o cálculo do dobro do número
    $resultado = $numero * 2;
    return $resultado;
  }

  echo "O dobro de 2 é " . calcula_dobro(2) . "<br>";
  echo "O dobro de 50 é " . calcula_dobro(50);

?>

Funções

Não há limite de argumentos que podemos passar para uma função.

Vamos criar, por exemplo, uma função que calcula a potência de um número elevado à outro.

<?php
  // exercicio2.php

  function potencia($base, $expoente) {
    // Faz o cálculo da potência
    $resultado = $base ** $expoente;
    return $resultado;
  }

  echo "2 elevado à 3 é " . potencia(2, 3) . "<br>";
  echo "10 elevado à 2 é " . potencia(10, 2);

?>

Funções

Podemos usar também funções dentro de funções. Por exemplo, vamos calcular a raiz quadrada de um número reutilizando a função de calcular potência.

<?php
  // exercicio3.php
  function potencia($base, $expoente) {
    $resultado = $base ** $expoente;
    return $resultado;
  }

  function raiz_quadrada($numero) {
    // Faz o cálculo da raiz quadrada do número
    $resultado = potencia($numero, 1/2);
    return $resultado;
  }

  echo "A raiz quadrada de 4 é " . raiz_quadrada(4) . "<br>";
  echo "A raiz quadrada de 100 é " . raiz_quadrada(100);

?>
