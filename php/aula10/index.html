Atividade

Problema: Pedrinho quer um sistema para sua locadora de filmes. Ele gostaria de armazenar os nomes, preços de aluguel, e sinopses dos filmes em um banco de dados. Como o preço pode variar com o tempo, mais filmes chegam a todo momento, e às vezes seus clientes perdem os filmes, Pedrinho gostaria de poder fazer isso usando formulários em um sistema web.

Instruções

  • Crie um sistema com PHP e MySQL para resolver o problema de Pedrinho. Utilize esse esqueleto de modelo para lhe ajudar a começar.

  • O banco de dados deverá ser nomeado como bdLocadora.
  • A tabela de filmes deverá ser nomeada filme e deverá conter os campos id, titulo, precoLocacao, e sinopse.
